Road Curve Editor

Trace a reference image, shape Catmull-Rom road splines, then export JSON for the Roblox road editor plugin.

Splines

Append points by clicking empty canvas space. Drag points to refine the layout.

Curves

Junctions

This tab enables junction editing for adding, moving, deleting, and shaping simple junctions. Radius clips the incoming roads and defines which control points move with the junction.

Trace Image

Import a flat reference image and align it in world X/Z space.

Session JSON

Autosaves locally after edits. Export JSON when you want a portable copy for Studio or another browser.

Canvas Controls

  • Left click empty space: append point to active spline
  • Left drag point: move point
  • Split Curve: select an interior point on an open curve
  • Junctions tab: click an existing curve point to add
  • Junctions tab: drag center to move the junction and grouped points
  • Junctions tab: drag radius ring to change road entrance clipping, Alt-click to delete
  • Right drag or middle drag: pan
  • Mouse wheel: zoom
